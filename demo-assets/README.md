# Demo Assets για GetFit App - App Store Review

Αυτό το φάκελο περιέχει όλα τα απαραίτητα demo assets για την απάντηση στο App Store review της Apple.

## 📁 Περιεχόμενα

### 1. QR Code Demo Assets
- **`create-demo-qr-codes.html`** - HTML page που δημιουργεί 4 demo QR codes
- Ανοίξτε το αρχείο στον browser για να δημιουργήσετε και κατεβάσετε τα QR codes

### 2. AR Marker Demo Assets  
- **`create-demo-ar-markers.html`** - HTML page που δημιουργεί 4 demo AR markers
- Ανοίξτε το αρχείο στον browser για να δημιουργήσετε και κατεβάσετε τα AR markers

### 3. App Store Response
- **`app-store-review-response.md`** - Πλήρης απάντηση για το App Store Connect
- Αντιγράψτε το περιεχόμενο στο App Review Information section

## 🚀 Πώς να τα χρησιμοποιήσετε

### Βήμα 1: Δημιουργία Demo Assets
1. Ανοίξτε `create-demo-qr-codes.html` στον browser
2. Κάντε κλικ "Κατέβασμα Όλων των QR Codes"
3. Ανοίξτε `create-demo-ar-markers.html` στον browser  
4. Κάντε κλικ "Κατέβασμα Όλων των AR Markers"

### Βήμα 2: Απάντηση στο App Store Connect
1. Συνδεθείτε στο [App Store Connect](https://appstoreconnect.apple.com/)
2. Επιλέξτε την εφαρμογή "GetFit_Skg"
3. Κάντε κλικ στην έκδοση 1.0
4. Πηγαίνετε στο "App Review Information"
5. Στο πεδίο "Notes" αντιγράψτε το περιεχόμενο από `app-store-review-response.md`
6. Ανέβασε τα QR codes και AR markers ως συνημμένα
7. Κάντε κλικ "Save"
8. Υποβάλετε ξανά για review
